# AoC 2024

This year it will be in [zig](https://ziglang.org/).

To build the code for a given day:

```shell
zig build -Dday=<X>
```

The executable will be located at `./zig-out/bin/AoC<X>`.

One can also build and run in one command:

```shell
zig build run -Dday=<X>
```
